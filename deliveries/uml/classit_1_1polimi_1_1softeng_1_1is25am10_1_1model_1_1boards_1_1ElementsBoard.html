Detailed Description

Used as a superclass by the classes AstronautBoard, AlienBoard, BatteryBoard and GoodsBoard. It provides a framework for those classes which place units above certain tiles. Contains some functions such as constructor, get methods, and some basic management such as removing units or moving units between tiles.
