Detailed Description

This class gives an interface between the model and the view. It shows available methods to the clients which won't affect directly the model, but this class will call the methods of the model. Most of the methods can be called on different models, each match has its own model, this is why each of them has the param name to identify the specific model through the name of the player calling the method.
